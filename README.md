## Setup

### MacOS
```
  brew install yarn
  yarn
  yarn start
```

### Linux
```
  curl -o- -L https://yarnpkg.com/install.sh | bash 
  yarn
  yarn start
```

### Windows
```
  npm install --global yarn
  yarn
  yarn start
```
